EJERCICIO 3

¿Qué es DIV o para qué sirve? La etiqueta < div > en HTML se utiliza para crear secciones o agrupar contenidos en una página web. Es un elemento en bloque que puede contener texto y/o cero o más elementos en bloque o en línea. Se emplea para aplicar estilos CSS y estructurar el diseño de la página. Además, se puede utilizar para centrar bloques de contenido, crear efectos de columna y diferentes áreas de color. También es útil para crear bloques de diseño CSS y colocarlos en el documento. La etiqueta < div > es fundamental para la estructura y organización de una página web.

¿Para qué sirve li a:hover? La pseudo-clase :hover en CSS se utiliza para aplicar estilos a un elemento cuando el cursor del mouse se sitúa sobre él. Por ejemplo, al usar li a:hover, se pueden definir estilos que se aplicarán cuando el cursor esté sobre un enlace dentro de un elemento de lista (li). Esto permite cambiar propiedades como el color, el fondo o la decoración del texto cuando el usuario interactúa con el elemento. Es una herramienta útil para mejorar la experiencia del usuario al proporcionar retroalimentación visual sobre la interactividad de los elementos en una página web.
